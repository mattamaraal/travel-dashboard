# ✈️ Travel Dashboard

Travel Dashboard é um dashboard para planejamento financeiro de viagens desenvolvido em Google Sheets e Google Apps Script.

O projeto nasceu para auxiliar no planejamento de viagens internacionais, centralizando informações de orçamento, câmbio e despesas em um único lugar.

## Objetivos

- Automatizar atualização de cotações
- Facilitar acompanhamento do orçamento
- Centralizar planejamento da viagem
- Servir como projeto de portfólio utilizando Google Apps Script

## Principais funcionalidades

### v0.2.0

- Atualização automática da cotação EUR → BRL
- Histórico de cotações
- Cache de consultas
- Estatísticas de câmbio
    - Cotação atual
    - Máxima
    - Mínima
    - Média
    - Variação diária
- Dashboard em Google Sheets
- Atualização manual
- Atualização automática via Trigger

## Tecnologias
