# Setup

## Pré-requisitos

- NodeJS
- npm
- Google Account
- clasp

## Instalação

Clone o projeto.

```
git clone ...
```

Instale dependências.

```
npm install
```

Login.

```
clasp login
```

Baixe o projeto.

```
clasp clone <script-id>
```

Sincronize.

```
npm run push
```

Abra a planilha.

Execute.

```
initializeApplication()
```

Depois.

```
installTriggers()
```
